Installation (Linux)
--------------------

NOTE: **Arch** users can just install **Plainotepad** from the **AUR**: ([plainotepad](https://aur.archlinux.org/packages/plainotepad)) and skip next section.

Dependencies:

**Arch Linux**:

```
sudo pacman -S qt5pas
```

**Ubuntu**:

```
sudo apt install libqt5pas1
```

**Fedora**:

```
sudo dnf install qt5pas
```

Done, unpack and run:

```
cd ~/Downloads/
```
```
tar -xvf plainotepad-0.9.12.tar.gz
```
```
./plainotepad
```

To make available in your list of applications, got to **Help** > **Create Desktop Entry**.

Installation (Windows)
----------------------

Just run **plainotepad.exe** from unzip archive (it's portable app).

NOTE: your antivirus maybe marks the app as dangerous.
