Mouse

Ctrl+click Add/delete caret
Ctrl+drag Add caret with selection
Ctrl+wheel Zoom in/out
Ctrl+Shift+click Add carets column in several lines
Double-click Select clicked word
Triple-click Select entire line
Alt+drag Select column of text

Keyboard

Shift+Ctrl+R Toggle read-only mode
Ctrl+U Toggle word-wrap mode (off/window)
Ctrl+Up/Down Scroll line up/down
Ins Toggle insert/overwrite mode
Ctrl+Y Delete line(s)
Ctrl+D Duplicate line(s)
Alt+Up/Down Move selected lines up/down
Ctrl+K Delete to line end
Ctrl+Del Delete word next
Ctrl+BkSp Delete word previous
Shift+Alt+Down/Left/Right/Up Column select: down/left/right/up
Shift+Alt+PgDn/PgUp Column select: page down/up
Shift+Alt+Home/End Column select: to line begin/end
Ctrl+I Indent selection
Shift+Ctrl+I Unindent selection
Ctrl+L Expand selection to line, down
Esc Cancel carets + selection + drag-drop
Ctrl+Home/End Go to text begin/end
Ctrl+Left/Right Go to word previous/end
Shift+Ctrl+Home/End Go to text begin/end + select
Shift+Ctrl+Left/Right Go to word previous/end + select
